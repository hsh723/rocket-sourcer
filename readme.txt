해외 소싱 상품 분석 및 마진 계산을 위한 올인원 워드프레스 플러그인

== Description ==

Rocket Sourcer는 쿠팡 셀러를 위한 해외 소싱 및 상품 분석 플러그인입니다. 키워드 분석, 제품 분석, 마진 계산, 해외 소싱 기능을 제공하여 효율적인 상품 소싱을 지원합니다.

= 주요 기능 =

* **키워드 분석**
  * 검색량 분석
  * 경쟁 강도 분석
  * 트렌드 분석
  * 추천 점수 산출

* **제품 분석**
  * 기본 정보 분석
  * 수익성 분석
  * 판매 예측
  * 성공 가능성 평가

* **마진 계산**
  * 판매가 기준 마진 계산
  * 원가 기준 적정 판매가 계산
  * 쿠팡 수수료 자동 계산
  * 손익분기점 분석

* **해외 소싱**
  * 1688.com 상품 검색
  * AliExpress 상품 검색
  * 이미지 기반 유사 상품 검색
  * 배송비 계산

= 프리미엄 기능 =

* 실시간 가격 모니터링
* 경쟁사 분석 리포트
* 대량 키워드 분석
* API 액세스
* 우선 지원

== Installation ==

1. 워드프레스 관리자 페이지에서 '플러그인 > 새로추가'로 이동
2. 'Rocket Sourcer' 검색
3. '지금 설치' 클릭 후 '활성화' 클릭
4. 'Rocket Sourcer > 설정' 메뉴에서 API 키 입력
5. 기본 설정 구성 후 사용 시작

== Frequently Asked Questions ==

= API 키는 어디서 얻을 수 있나요? =

Rocket Sourcer 웹사이트(https://rocketsourcer.com)에서 계정을 생성하면 API 키를 발급받을 수 있습니다.

= 무료 버전과 프리미엄 버전의 차이는 무엇인가요? =

무료 버전은 기본적인 키워드 분석, 제품 분석, 마진 계산 기능을 제공합니다. 프리미엄 버전은 실시간 모니터링, 대량 분석, API 액세스 등 추가 기능을 제공합니다.

= 데이터는 얼마나 자주 업데이트되나요? =

키워드 데이터는 일 1회, 제품 데이터는 시간당 1회 업데이트됩니다. 프리미엄 사용자는 실시간 업데이트를 이용할 수 있습니다.

= 해외 배송비는 어떻게 계산되나요? =

실제 배송 업체 API와 연동하여 정확한 배송비를 계산합니다. 무게, 부피, 배송 방법에 따라 자동으로 계산됩니다.

= 여러 계정을 사용할 수 있나요? =

프리미엄 버전에서는 여러 계정을 추가하여 사용할 수 있습니다. 무료 버전은 하나의 계정만 사용 가능합니다.
