== Description ==

Rocket Sourcer는 쿠팡 로켓그로스 셀러들을 위한 강력한 소싱 추천 및 분석 도구입니다.

주요 기능:
* 인기 키워드 분석
* 제품 성과 분석
* 마진율 계산기
* 경쟁 분석
* 판매 예측

== Installation ==

1. 플러그인 파일을 `/wp-content/plugins/rocket-sourcer` 디렉토리에 업로드하거나 워드프레스 관리자 페이지에서 플러그인을 직접 설치하세요.
2. 워드프레스 관리자 페이지에서 플러그인을 활성화하세요.
3. Settings > Rocket Sourcer 메뉴에서 API 키를 설정하세요.

== Frequently Asked Questions ==

= API 키는 어디서 얻을 수 있나요? =

쿠팡 파트너스 개발자 센터(https://developers.coupangcorp.com)에서 발급받을 수 있습니다.

= 무료로 사용할 수 있나요? =

기본 기능은 무료로 제공되며, 고급 분석 기능은 프리미엄 버전에서 사용할 수 있습니다.
