# Rocket Sourcer

해외 소싱 상품 분석 및 마진 계산 워드프레스 플러그인

## 주요 기능

### 1. 키워드 분석
- 검색량 분석
- 경쟁 강도 분석
- 트렌드 분석
- 추천 점수 산출

### 2. 제품 분석
- 기본 정보 분석
- 수익성 분석
- 판매 예측
- 성공 가능성 평가

### 3. 마진 계산
- 판매가 기준 마진 계산
- 원가 기준 적정 판매가 계산
- 쿠팡 수수료 자동 계산
- 손익분기점 분석

### 4. 해외 소싱
- 1688.com 상품 검색
- AliExpress 상품 검색
- 이미지 기반 유사 상품 검색
- 배송비 계산

## 설치 방법

1. 플러그인 파일을 다운로드합니다.
2. 워드프레스 관리자 페이지에서 '플러그인 > 새로추가 > 플러그인 업로드'를 선택합니다.
3. 다운로드한 ZIP 파일을 선택하고 '지금 설치'를 클릭합니다.
4. 설치가 완료되면 '플러그인 활성화'를 클릭합니다.

## 시스템 요구사항

- PHP 7.4 이상
- MySQL 5.6 이상 또는 MariaDB 10.1 이상
- WordPress 5.8 이상
- PHP cURL 확장
- PHP JSON 확장
- PHP MySQL 확장

## 설정 방법

1. 워드프레스 관리자 페이지에서 'Rocket Sourcer' 메뉴를 클릭합니다.
2. 'API 설정' 탭에서 필요한 API 키를 입력합니다:
   - 1688.com API 키
   - AliExpress API 키
   - 배송비 계산 API 키
3. '일반 설정' 탭에서 기본 설정을 구성합니다:
   - 기본 마진율
   - 기본 배송비
   - 캐시 설정
   - 로그 설정

## 사용 방법

### 키워드 분석
1. '키워드 분석' 메뉴로 이동합니다.
2. 분석할 키워드를 입력합니다.
3. 카테고리를 선택합니다.
4. '분석 시작' 버튼을 클릭합니다.
5. 분석 결과를 확인합니다.

### 제품 분석
1. '제품 분석' 메뉴로 이동합니다.
2. 분석할 제품 URL을 입력합니다.
3. '분석 시작' 버튼을 클릭합니다.
4. 상세 분석 결과를 확인합니다.

### 마진 계산
1. '마진 계산기' 메뉴로 이동합니다.
2. 필요한 정보를 입력합니다:
   - 상품 원가
   - 판매가
   - 배송비
   - 쿠팡 수수료율
3. '계산하기' 버튼을 클릭합니다.
4. 계산 결과를 확인합니다.

## 문제 해결

### 로그 확인
- 'Rocket Sourcer > 로그' 메뉴에서 로그를 확인할 수 있습니다.
- 로그 레벨별로 필터링이 가능합니다.
- 로그 파일은 자동으로 순환됩니다.

### 일반적인 문제
1. API 연결 오류
   - API 키가 올바르게 입력되었는지 확인
   - API 서버 상태 확인
   - 네트워크 연결 상태 확인

2. 성능 문제
   - 캐시 설정 확인
   - 데이터베이스 최적화
   - PHP 메모리 제한 확인

3. 호환성 문제
   - PHP 버전 확인
   - WordPress 버전 확인
   - 테마/플러그인 충돌 확인
